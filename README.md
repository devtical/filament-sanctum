## Installation

You can install the package via composer:

```bash
composer require devtical/filament-sanctum
```

Config: 

```bash
php artisan vendor:publish --tag=filament-sanctum-config
```

Translations:

```bash
php artisan vendor:publish --tag=filament-sanctum-translations
```

## Testing

```bash
composer test
```
